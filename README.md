# OpenFrafsuallyLib.CSharp
A C# library to perform calculations related to Frametimes and framerates and FrameTime and AverageFrameRate models.

Usage in both static and dynamic linking is allowed.


## Usage
OpenFrafsually contains no GUI or UI code whatsoever and so implementation into a UI or GUI is left up to developers if they wish to integrate or use OpenFrafsuallyLib.CSharp in their own projects.
